# asepxyz-wrapper

Baileys command-handler wrapper, maintained by AsepXyz. Fork chain: this project
is a derivative of `@itsreimau/gktw`, which itself forked `@mengkodingan/ckptw`.

Now uses `asepxyz` (self-contained AsepXyz Baileys build) as its underlying
Baileys dependency instead of the original upstream fork, so installs no longer
depend on third-party GitHub accounts staying online.

## Installation

```bash
npm install github:AsepXyz12/asepxyz
```
